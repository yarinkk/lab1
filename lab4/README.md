Опис

Цей проєкт містить просту програму на Python для розрахунку вартості поїздки таксі залежно від відстані.
Як запустити проєкт

Клонувати репозиторій:

git clone https://github.com/yarinkk/lab1.git

Перейти в папку проєкту:

cd lab1/lab4

Створити та активувати віртуальне середовище:

python -m venv venv

venv\Scripts\activate

Встановит залежності:
pip install -r requirements.txt

Запустити програму:
python taxi_service.py
